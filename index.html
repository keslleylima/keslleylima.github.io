The use of regression analysis to predict prime path coverage, which is a powerful test coverage criteria to aid the tester in assessing the quality of the test suite, was investigated in this key study (Published here). The Knowledge Discovery in Database approach was employed, and a dataset was built from 9 real-world projects to develop 3 regression models for prime-path prediction.

It was compared with four different machine learning algorithms, and a fine-grained feature analysis was carried out in order to identify the features with the most impact on prediction accuracy.The experimental findings reveal that the developed models achieved satisfactory prediction performance utilizing only five source code metrics and one basic coverage measure as input data.
